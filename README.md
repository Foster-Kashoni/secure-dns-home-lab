# Secure DNS Home Lab

This repository contains a documented home lab project implementing:

- Pi-hole as a network-wide DNS sinkhole
- Unbound as a recursive DNS resolver
- HTTPS-secured admin interface using Let's Encrypt
- Hosted on Raspberry Pi 5
